# Yasir Aziz - Professional Portfolio Website

A modern, responsive single-page portfolio website built with React.js showcasing my professional experience, education, skills, and projects.

## 🚀 Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Elegant fade-in and slide-up animations for enhanced user experience
- **Single Page Application**: Smooth scroll navigation between sections
- **Modern UI**: Clean, professional design with gradient accents and subtle shadows
- **Interactive Elements**: Hover effects and transitions on interactive components

## 📋 Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About Me**: Professional summary highlighting background and expertise
3. **Skills**: Categorized display of technical and soft skills
4. **Experience**: Timeline view of professional work history
5. **Projects**: Showcase of academic and personal data science projects
6. **Education**: Academic credentials and honors
7. **Contact**: Multiple contact methods and portfolio links

## 🛠️ Technologies Used

- **React.js 18.2.0**: Frontend framework
- **CSS3**: Styling with CSS Modules approach
- **React Hooks**: useState, useEffect, useRef for state management
- **Intersection Observer API**: For scroll-based animations
- **Google Fonts (Inter)**: Modern, clean typography

## 📦 Installation & Setup

1. **Clone or download this repository**

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Open your browser** and navigate to `http://localhost:3000`

## 🏗️ Build for Production

To create an optimized production build:

```bash
npm run build
```

This creates a `build` folder with optimized files ready for deployment.

## 🎨 Design Choices

- **Color Scheme**: Professional blue gradient (primary) with green accent (secondary)
