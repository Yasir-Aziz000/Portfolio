## 🎨 Easy Customization

### Change Colors
Edit `src/index.css` variables:
```css
--primary-color: #2563eb;  /* Your primary color */
--secondary-color: #10b981; /* Accent color */
```

### Update Content
Each section is in its own component file - just edit the relevant `.js` file!

### Add More Projects
Edit `src/components/Projects.js` and add to the `projects` array.
